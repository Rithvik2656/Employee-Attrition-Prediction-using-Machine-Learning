# 🚀 Employee Attrition Prediction System

A Machine Learning web application that predicts whether an employee is likely to leave the company.

---

## 🔥 Features
- 📊 Predict employee attrition
- 🤖 ML model (Logistic Regression + Decision Tree + Random Forest)
- 🌐 Flask web application
- 🎨 Modern UI (Dashboard + Prediction page)

---

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML, CSS, JS

---

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
python train_model.py
python app.py
